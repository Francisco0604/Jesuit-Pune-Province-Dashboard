# PROJECT_MAP.txt - Jesuit Pune Province Activity Map

## Project Overview
A high-performance web dashboard for the Jesuit Pune Province to visualize activity centers (Parish, Education, Social Justice, TDSS Partners).

## Tech Stack
- **Framework:** Next.js 15+ (App Router)
- **Language:** TypeScript
- **Database:** Supabase (PostgreSQL)
- **Styling:** Tailwind CSS 4
- **Mapping:** react-leaflet, react-leaflet-cluster
- **State Management:** TanStack Query (React Query)
- **Icons:** lucide-react

## Directory Structure
- `/app`: Next.js pages and layouts (Routes)
- `/components`: UI elements (Map, Drawer, Sidebar)
- `/lib`: Database and external service connections (Supabase)
- `/utils`: Data normalization and helper functions (dataProcessor.ts)
- `/types`: TypeScript interfaces for your data
- `/public`: Static assets (Logos, maps, documentation PDFs)

## File Registry
- `PROJECT_MAP.txt`: This file (Project blueprint and documentation)
- `Instruction.txt`: Original project requirements (in /legacy)
- `lib/supabase.ts`: Supabase client and fetch functions
- `utils/dataProcessor.ts`: Data normalization utility for JSON/GeoJSON/QGIS
- `types/index.ts`: TypeScript interfaces for Center and RawData
- `components/Providers.tsx`: React Query and other context providers
- `components/MapComponent.tsx`: Leaflet map implementation with marker clustering
- `components/CenterDrawer.tsx`: Detail view for centers (Right sidebar/Bottom sheet)
- `components/FilterSidebar.tsx`: Searchable list and category filters (Left sidebar)
- `src/app/page.tsx`: Main dashboard entry point
- `src/app/globals.css`: Ecclesiastical theme styling and glassmorphism
- `src/app/layout.tsx`: Root layout with EB Garamond and Inter fonts
- `src/app/admin/page.tsx`: Admin dashboard for CRUD and CSV import
- `public/data/sangamner_parish.json`: Extracted data from the St Mary's Parish PDF
- `public/data/villages.json`: Sample village data
- `public/data/sample.geojson`: Sample GeoJSON data

## 🛠 Features Implemented
- **Interactive Map:** Optimized with lazy loading, custom MapPin icons, and Marker Clustering.
- **Data Normalization:** Robust utility handling JSON, GeoJSON, and QGIS formats.
- **Mobile-First UX:** Sidebars transform into Bottom Sheets on small screens for touch-native feel.
- **Ecclesiastical Theme:** Custom Tailwind configuration with parchment and terracotta palette.
- **Data Enrichment:** Integrated 40+ village records extracted from original parish PDF.

## Setup & Deployment
- `npm install`: Install dependencies
- `npm run dev`: Start development server
- Environment variables: `.env.local` (Requires Supabase URL and Anon Key)
